# Abstraction Alignment

This repo contains code for Abstraction Alignment: Measuring, Interpreting, and Learning from Models’ Hierarchical Relationships. To run the interface and explor the CIFAR-100 and MIMIC-III abstraction alignment experiments from the paper:
1. Clone this repo
2. Download the data from the supplementary submission files and unzip them into `data/`. You should have `data/cifar/` and `ata/mimic/`.
3. Run `npm install` and `npm run dev`
4. Open `localhost:5173` and explore!
